# Plan de Acción – Frontend (React + Vite + Tailwind + react-icons)

## Requisitos
- Node 18+

## Instalación
```bash
npm i
npm run dev
```
Crea/ajusta `.env`:
```
VITE_API_URL=http://localhost:8000
```

## Características
- Header
- Form de creación de plan + tabla de listado
- Estilos con Tailwind + react-icons
